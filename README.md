## 🚀 How to Use

### Option 1 — Open Directly (Easiest)
1. Download `nova-final.html`
2. Open it in any modern browser (Chrome, Firefox, Edge, Safari)
3. Enter your Groq API key on the setup screen
4. Start chatting!

### Option 2 — GitHub Pages
1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your NOVA will be live at `https://yourusername.github.io/nova-ai/`

### Option 3 — Any Static Host
Upload `nova-final.html` to:
- **Netlify** — drag and drop the file at [netlify.com/drop](https://app.netlify.com/drop)
- **Vercel** — deploy via CLI or dashboard
- **Cloudflare Pages** — connect your GitHub repo

## 🔑 Get Your Free Groq API Key

1. Visit [console.groq.com](https://console.groq.com)
2. Sign up (free, no credit card needed)
3. Go to **API Keys → Create API Key**
4. Copy the key — it starts with `gsk_`
5. Paste it into NOVA's setup screen

> **Your key is stored only in your browser session.** It is never sent anywhere except directly to Groq's API. NOVA has no server, no database, and no tracking.
